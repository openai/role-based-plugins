# Prompt Patterns

## Executive Visual One-Pager

```text
Create a leadership-ready one-page chart visual. Use restrained premium business design, strong hierarchy, crisp typography, subtle gridlines, and a clean footnote zone. Keep the chart central and preserve all data exactly.
```

## Comparison Proof Card

```text
Create an evidence-oriented comparison card. Emphasize the stated comparison and proof hierarchy without adding claims. Use controlled badges or proof framing only outside the plot area.
```

## Creative Learning Memo

```text
Create a creative-results memo around the chart. Add tasteful learning-note framing, review cues, or checklist elements outside the plot area. Do not hide any chart content.
```

## Landing Page Hero

```text
Create a first-viewport hero visual with the chart on a white rounded card over a premium background or supplied texture. Keep the chart fully readable and do not add CTA copy unless the user asks.
```

## Storyboard / Proof Scene

```text
Create a single-image proof storyboard where the source chart remains one intact central evidence panel. Add story framing around the chart only. Do not split the chart into inaccurate panels.
```

## Channel Format Wall

```text
Create a production-review wall treatment. Add safe zones, crop rails, format outlines, or export markers around the chart. Do not duplicate the chart or create extra chart data.
```

## Drawing Style

```text
Create a hand-drawn chart treatment with ink outlines, light cross-hatching, notebook texture, and carefully legible handwritten-style labels. Preserve all values and labels.
```

## 3D Style

```text
Create a soft premium 3D chart treatment with rounded tactile bars, realistic shadows, and gentle depth. Do not let perspective distort perceived bar heights or add decorative columns.
```

## Characters / Mini Me

```text
Create tiny 3D or illustrated characters around the chart margins only. Characters may point, wave, or react from whitespace, but must not touch, cover, sit on, climb on, or overlap real bars, labels, values, annotations, or axes. Do not add bar-like props.
```

## Minimalist

```text
Create a minimalist chart treatment with spacious layout, rounded bars, restrained gridlines, soft neutral panel, careful typography, and no extra decoration.
```
