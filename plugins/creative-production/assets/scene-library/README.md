# Scene Library

Hydratable reusable Creative Production scene packs live here.

Run `python plugins/creative-production/scripts/hydrate_assets.py --check` before reading packs. If assets are missing, hydrate from the Drive upload payload with `--source-dir`.

Scene packs define commercial environments where a supplied product, service, venue, offer, or business asset can be placed. Skills should consume this library instead of keeping private copies in their own `references/` folders.

Current packs:

- `packs/business-point-of-sale-scenes.json`: hydratable 25 slot-based business, retail, service, customer, demo, pickup, and handoff scene archetypes.
